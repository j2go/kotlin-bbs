# 简介
**kotlin bbs**

使用 layui 的社区模板，springboot ，kotlin 语言实现，目标是尽量少的代码

*简洁而不简单*

# 安装
1. 环境 jdk17, mysql8
2. `git clone https://github.com/j2go/kotlin-bbs.git`
3. `./gradlew build`
4. 配置数据库连接(application.properties)
5. `./gradlew bootRun`

# 技术

- Kotlin
- Spring Boot2
- Spring Security
- Freemarker
- Layui.js
